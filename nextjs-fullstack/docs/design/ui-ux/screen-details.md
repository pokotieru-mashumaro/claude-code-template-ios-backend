## AUTH-001: ログイン画面

### 概要
- パス: `/login`
- 認証: 不要

### 構成要素
- メールアドレス入力フィールド
- パスワード入力フィールド
- ログインボタン
- 新規登録リンク
- パスワードリセットリンク

### 動作
1. メールアドレスとパスワードを入力
2. ログインボタンをクリック
3. 認証成功時: ダッシュボードへリダイレクト
4. 認証失敗時: エラーメッセージを表示

## MAIN-002: ダッシュボード

### 概要
- パス: `/dashboard`
- 認証: 必要

### 構成要素
- ヘッダーナビゲーション
- メインコンテンツエリア
- サイドバー（必要に応じて）

### 動作
1. ページ読み込み時にユーザー情報を取得
2. ダッシュボードコンテンツを表示
