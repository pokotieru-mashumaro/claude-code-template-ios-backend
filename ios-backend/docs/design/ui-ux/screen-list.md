# 画面一覧

## 画面構成

### 認証フロー

| 画面名 | 概要 | 主要機能 |
|-------|------|---------|
| スプラッシュ画面 | アプリ起動時の初期画面 | - ローディング表示<br>- 認証状態確認 |
| ログイン画面 | ログイン | - メール・パスワード入力<br>- ログインボタン<br>- 新規登録へのリンク |
| 新規登録画面 | 新規ユーザー登録 | - メール・パスワード・名前入力<br>- 利用規約同意<br>- 登録ボタン |

---

### メインフロー

| 画面名 | 概要 | 主要機能 |
|-------|------|---------|
| ホーム画面 | メイン画面 | - [プロジェクト固有の機能] |
| プロフィール画面 | ユーザープロフィール表示・編集 | - プロフィール情報表示<br>- 編集ボタン |
| 設定画面 | アプリ設定 | - アカウント設定<br>- 通知設定<br>- ログアウト |

---

### その他

| 画面名 | 概要 | 主要機能 |
|-------|------|---------|
| エラー画面 | エラー表示 | - エラーメッセージ表示<br>- リトライボタン |
| ローディング画面 | ローディング表示 | - インジケーター表示 |
