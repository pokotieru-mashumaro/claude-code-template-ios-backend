## ログイン画面

### 概要
ユーザーがメールアドレスとパスワードでログインする画面。

### UIコンポーネント

#### 1. タイトル
- テキスト: "ログイン"
- フォント: システムフォント（Bold、24pt）

#### 2. メールアドレス入力フィールド
- プレースホルダー: "メールアドレス"
- キーボードタイプ: Email
- バリデーション: メール形式

#### 3. パスワード入力フィールド
- プレースホルダー: "パスワード"
- セキュア入力: ON
- バリデーション: 8文字以上

#### 4. ログインボタン
- テキスト: "ログイン"
- 状態:
  - デフォルト: 有効（背景色: プライマリカラー）
  - 無効: 入力が不正な場合（背景色: グレー）
  - ローディング: 処理中（インジケーター表示）

#### 5. 新規登録リンク
- テキスト: "アカウントをお持ちでない方はこちら"
- タップ時: 新規登録画面へ遷移

### ユーザーフロー

1. ユーザーがメールアドレスとパスワードを入力
2. ログインボタンをタップ
3. バリデーションチェック
4. APIリクエスト送信
5. 成功 → ホーム画面へ遷移
6. 失敗 → エラーメッセージ表示

### エラーメッセージ

| エラーケース | メッセージ |
|------------|----------|
| メール形式エラー | "正しいメールアドレスを入力してください" |
| パスワード長エラー | "パスワードは8文字以上で入力してください" |
| 認証エラー | "メールアドレスまたはパスワードが正しくありません" |
| ネットワークエラー | "通信エラーが発生しました。再度お試しください" |

## 新規登録画面

### 概要
新しいユーザーがアカウントを作成する画面。

### UIコンポーネント

#### 1. タイトル
- テキスト: "新規登録"
- フォント: システムフォント（Bold、24pt）

#### 2. 名前入力フィールド
- プレースホルダー: "名前"
- バリデーション: 1文字以上、50文字以内

#### 3. メールアドレス入力フィールド
- プレースホルダー: "メールアドレス"
- キーボードタイプ: Email
- バリデーション: メール形式

#### 4. パスワード入力フィールド
- プレースホルダー: "パスワード（8文字以上）"
- セキュア入力: ON
- バリデーション: 8文字以上

#### 5. 利用規約同意チェックボックス
- テキスト: "利用規約に同意する"
- リンク: 利用規約ページへ

#### 6. 登録ボタン
- テキスト: "登録"
- 状態:
  - デフォルト: 有効（背景色: プライマリカラー）
  - 無効: 入力が不正 or 利用規約未同意（背景色: グレー）
  - ローディング: 処理中（インジケーター表示）

#### 7. ログインリンク
- テキスト: "既にアカウントをお持ちの方はこちら"
- タップ時: ログイン画面へ遷移

### ユーザーフロー

1. ユーザーが名前、メールアドレス、パスワードを入力
2. 利用規約に同意
3. 登録ボタンをタップ
4. バリデーションチェック
5. APIリクエスト送信
6. 成功 → ホーム画面へ遷移
7. 失敗 → エラーメッセージ表示

### エラーメッセージ

| エラーケース | メッセージ |
|------------|----------|
| 名前未入力 | "名前を入力してください" |
| メール形式エラー | "正しいメールアドレスを入力してください" |
| パスワード長エラー | "パスワードは8文字以上で入力してください" |
| メール重複エラー | "このメールアドレスは既に登録されています" |
| ネットワークエラー | "通信エラーが発生しました。再度お試しください" |
